# НМТ Аналітика

Статичний React SPA для аналізу результатів НМТ-тестів.  
Завантажте CSV-експорт з платформи — отримайте рейтинг учнів, журнал, складні питання та графіки. Жодного бекенду, жодної реєстрації.

## Можливості

- **Рейтинг учнів** — сортування, пошук, детальний профіль з розбивкою по питаннях
- **Журнал** — матриця учень × предмет зі шкальним і сирим балом
- **Питання** — всі питання відсортовані від найскладнішого, відповіді кожного учня
- **Графіки** — розподіл NMT, тривалість vs результат, порівняння класів
- **Фільтри** — по предмету і класу, KPI оновлюються миттєво

## Запуск локально

```bash
npm install
npm run dev
```

Відкрийте http://localhost:5173

## Збірка

```bash
npm run build   # результат у папці dist/
npm run preview # перегляд зібраної версії
```

## Розгортання

**Netlify** (найпростіше):
1. Запустіть `npm run build`
2. Перетягніть папку `dist/` на [netlify.com/drop](https://app.netlify.com/drop)

**Vercel**:
1. Підключіть GitHub-репозиторій на [vercel.com](https://vercel.com)
2. Framework preset: **Vite** — Vercel визначить автоматично

**GitHub Pages**:
1. Встановіть пакет: `npm install -D gh-pages`
2. Додайте до `vite.config.js`: `base: '/назва-репозиторію/'`
3. Додайте до `package.json` → `scripts`: `"deploy": "gh-pages -d dist"`
4. `npm run build && npm run deploy`

## Формат CSV

Файл має містити колонки:

```
Email, Студент, Клас, Предмет (Блок), Початок, Кінець,
Бал за блок (Raw), Загальний Raw, Загальний NMT,
Питання, Тип, Відповідь студента, Правильна відповідь, Балів за питання
```

Колонка `Клас` — опціональна. Кодування UTF-8.
